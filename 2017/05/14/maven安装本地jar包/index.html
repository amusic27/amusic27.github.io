maven安装本地jar包

发表于 2017-05-14 | 分类于 maven | | 阅读次数

  • 将要安装的jar包放到一个目录，如 ojdbc14.jar
    目录为：f:/jar/

  • cd到jar包所在目录 执行如下命令

    mvn install:install-file -DgroupId=com.oracle -DartifactId=ojdbc14 -Dversion=10.2.0.4.0 -Dpackaging=jar -Dfile=ojdbc14.jar
